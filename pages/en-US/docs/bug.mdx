# Welcome to Report Bugs

We greatly welcome feedback and suggestions from community users. If you find any bugs during use, you can submit issues through the [NEW ISSUE](https://github.com/CodePhiliaX/easyexcel-plus/issues/new/choose) page.

---

## Recommended Process for Submitting Issues

To ensure your feedback can be processed quickly, please follow these steps:

### 1. **Search Existing Issues**
Before submitting an issue, please check if your problem has already been reported by other users:
- Visit the [Issues List](https://github.com/CodePhiliaX/easyexcel-plus/issues) and search for keywords.
- If you find a similar issue, please **comment and add detailed information** to that issue instead of creating a new one.

This helps avoid duplicate issues and improves the efficiency of issue handling.

---

### 2. **Create New Issue**
If you confirm your issue hasn't been reported, please submit a new issue through the [NEW ISSUE](https://github.com/CodePhiliaX/easyexcel-plus/issues/new/choose) page.

- **Please use the issue template** to ensure you provide complete information, such as:
  - Environment where the issue occurred (system, version, etc.).
  - Steps to reproduce.
  - Error logs or screenshots.

---

Thank you for your contribution to improving the project quality! Every piece of feedback is our motivation to move forward.
