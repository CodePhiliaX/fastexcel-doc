## Convert Excel to PDF

### Overview
FastExcel supports converting Excel files to PDF files. FastExcel converts Excel to PDF using Apache POI and itext-pdf as the underlying dependencies. Due to the license of itext-pdf, please ensure your usage complies with the itext-pdf license.
In the future, FastExcel will support more PDF conversion features. Replace itext-pdf.

#### Sample Object
```java
FastExcel.convertToPdf(new File("excelFile"), new File("pdfFile"), null, null);
```
