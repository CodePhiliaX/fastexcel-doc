## Convert Excel to PDF

### Overview
FastExcel supports converting Excel files to PDF files. FastExcel relies on Apache POI and itext-pdf for converting Excel to PDF. Please ensure that your usage complies with the license of itext-pdf.
In the future, FastExcel will support more PDF conversion features. Replace itext-pdf.

#### Sample Object
```java
FastExcel.convertToPdf(new File("excelFile"), new File("pdfFile"), null, null);
```
