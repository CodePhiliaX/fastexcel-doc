## **Simple Fill**

### Overview
Fill data into Excel based on template files using objects or Maps.

#### Sample Object
```java
@Getter
@Setter
@EqualsAndHashCode
public class FillData {
    private String name;
    private double number;
    private Date date;
}
```

#### Sample Code
```java
@Test
public void simpleFill() {
    String templateFileName = "path/to/simple.xlsx";

    // Method 1: Fill using object
    FillData fillData = new FillData();
    fillData.setName("John");
    fillData.setNumber(5.2);
    FastExcel.write("simpleFill.xlsx")
        .withTemplate(templateFileName)
        .sheet()
        .doFill(fillData);

    // Method 2: Fill using Map
    Map<String, Object> map = new HashMap<>();
    map.put("name", "John");
    map.put("number", 5.2);
    FastExcel.write("simpleFillMap.xlsx")
        .withTemplate(templateFileName)
        .sheet()
        .doFill(map);
}
```
