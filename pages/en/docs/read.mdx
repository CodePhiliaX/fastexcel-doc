# Installation

The following table shows the minimum Java version requirements for different versions of EasyExcel-Plus:

| Version | JDK Version Support | Notes |
|---------|:------------------:|-------|
| 4.0.0+ | jdk8 - jdk21 | Current master branch, fully compatible with easyexcel |
| 5.0.0+ | jdk8 - jdk21 | Next generation version, removes poi dependency |

We strongly recommend using the latest version of EasyExcel-Plus, as performance optimizations, bug fixes, and new features will make your usage more convenient.

> Since EasyExcel-Plus uses poi as its foundation package, if you are using version 4+ and your project already has poi-related components, you need to manually exclude poi-related jar packages.

If you use Maven for project building, add the following configuration to your `pom.xml` file:
```xml
<dependency>
    <groupId>com.alibaba</groupId>
    <artifactId>easyexcel</artifactId>
    <version>4.0.3</version>
</dependency>
```

If you use Gradle for project building, add the following configuration to your `build.gradle` file:
```gradle
dependencies {
    implementation 'com.alibaba:easyexcel:4.0.3'
}
```

# Updates

Visit [easyexcel-plus](https://github.com/alibaba/easyexcel) to get the latest version.

# Reading Excel Files

Here's an example of reading an Excel document:
```java
// Implement ReadListener interface to set up data reading operations
public class DemoDataListener implements ReadListener<DemoData> {
    @Override
    public void invoke(DemoData data, AnalysisContext context) {
        System.out.println("Parsed one row of data: " + JSON.toJSONString(data));
    }

    @Override
    public void doAfterAllAnalysed(AnalysisContext context) {
        System.out.println("All data parsing completed!");
    }
}

public static void main(String[] args) {
    String fileName = "demo.xlsx";
    // Read Excel file
    EasyExcel.read(fileName, DemoData.class, new DemoDataListener()).sheet().doRead();
}
```

# Creating Excel Files

Here's a simple example of creating an Excel document:
```java
// Example data class
public class DemoData {
    @ExcelProperty("String Title")
    private String string;
    @ExcelProperty("Date Title")
    private Date date;
    @ExcelProperty("Number Title")
    private Double doubleData;
    @ExcelIgnore
    private String ignore;
}

// Prepare data to write
private static List<DemoData> data() {
    List<DemoData> list = new ArrayList<>();
    for (int i = 0; i < 10; i++) {
        DemoData data = new DemoData();
        data.setString("String " + i);
        data.setDate(new Date());
        data.setDoubleData(0.56);
        list.add(data);
    }
    return list;
}

public static void main(String[] args) {
    String fileName = "demo.xlsx";
    // Create a sheet named "Template" and write data
    EasyExcel.write(fileName, DemoData.class).sheet("Template").doWrite(data());
}
```
