# お問い合わせ

## コミュニティサポート

FastExcelを使用している際に問題が発生した場合やご意見がある場合は、以下の方法でお問い合わせください：

### GitHub

- [GitHubリポジトリ](https://github.com/CodePhiliaX/fastexcel)をご覧ください
- [Issueを提出](https://github.com/CodePhiliaX/fastexcel/issues)
- [Discussionsに参加](https://github.com/CodePhiliaX/fastexcel/discussions)

### 貢献ガイドライン

FastExcelにコードを貢献したい場合は、[貢献ガイドライン](/zh-CN/docs/contribution)をご覧ください。

### 最新情報

作者「程序员小懒」の公式アカウントをフォローして、最新の技術情報やアップデートを入手してください。
