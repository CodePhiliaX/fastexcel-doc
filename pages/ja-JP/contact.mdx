# お問い合わせ

## コミュニティサポート

FastExcelの使用中に問題が発生した場合や、ご提案がございましたら、以下の方法でお気軽にお問い合わせください：

### GitHub

- [GitHubリポジトリ](https://github.com/CodePhiliaX/fastexcel)にアクセス
- [Issue](https://github.com/CodePhiliaX/fastexcel/issues)を提出
- [Discussions](https://github.com/CodePhiliaX/fastexcel/discussions)に参加

### コントリビューションガイド
FastExcelにコードを提供したい場合は、[コントリビューションガイド](/zh-CN/docs/contribution)をご覧ください。

### 最新情報
著者「プログラマー小懒」の公式WeChat（微信）アカウントをフォローして、より多くの技術情報や最新のアップデートを入手してください。
