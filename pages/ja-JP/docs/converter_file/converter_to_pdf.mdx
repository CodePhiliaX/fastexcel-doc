## ExcelをPDFに変換する

### 概要
FastExcelはExcelファイルをPDFファイルに変換する機能をサポートしています。FastExcelはExcelをPDFに変換する際にApache POIとitext-pdfに依存しています。itext-pdfのライセンスに制限があるため、ご利用の際はitext-pdfのライセンスに準拠していることを確認してください。
今後、FastExcelはより多くのPDF変換機能をサポートし、itext-pdfを置き換える予定です。

#### サンプルコード
```java
FastExcel.convertToPdf(new File("excelFile"),new File("pdfFile"),null,null);
```
